Getting Started¶

Dependencies¶

The plasma_properties package requires the use of Python 3. If you do not have Python 3 installed on your machine, you can install the Anaconda Distribution which will give you access to Python 3 and many additional packages built in Python 3. Click here to obtain the Anaconda distribution of Python.

Access¶

Currently you can gain access to the plasma_properties package by using pip or by cloning the GitHub repository.

To Install from pip (Recommended)¶

Open a terminal and type the following command:

$ pip install plasma-properties-package

Warning

This package is in its early stages and going through a lot of development. Please check your version for bugfixes and new features. To upgrade your version using pip:

$ pip install plasma-properties-package --upgrade

To Download from GitHub¶

Open a terminal and type the following command:

$ git clone git@github.com:lukestanek/plasma_properties.git
